# 9.2-1
i < k のとき q-1 ≧ l であることと、i > k のとき r ≧ q+1 であることを示せばよい。  
l ≦ q ≦ r , k = q-l+1 , 1 ≦ i ≦ r-l+1 であることに注意して、  
i < k = q-l+1 ⇔ q-l+1 > i ⇔ q-1 ≧ i+l-1 ⇔ q-1 ≧ r ≧ l となり成立。  
i > k = q-l+1 ⇔ i+l-1 ≧ q+1 ⇔ r ≧ l ≧ q+1 となり成立。
